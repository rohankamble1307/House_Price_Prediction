# 🏠 House Price Prediction

## 📌 Overview

This project aims to predict house prices using Machine Learning techniques. It uses various housing features such as area, quality, and construction details to estimate the final sale price of a house.

---

## 🚀 Features

* Data Cleaning & Preprocessing
* Missing Value Handling (Mean & Mode)
* One-Hot Encoding for categorical data
* Exploratory Data Analysis (EDA)
* Correlation Heatmap Visualization
* Model Training & Evaluation

---

## 📂 Dataset

* Dataset contains multiple features related to houses
* Target Variable: **SalePrice**

---

## ⚙️ Technologies Used

* Python
* Pandas
* NumPy
* Matplotlib
* Seaborn
* Scikit-learn

---

## 🧠 Models Used

### 🔹 Linear Regression

* R² Score: **0.65**
* Basic model for linear relationships

### 🔹 Random Forest Regressor

* R² Score: **0.89**
* Best performing model
* Handles complex and non-linear relationships

---

## 📊 Evaluation Metrics

* Mean Absolute Error (MAE)
* R² Score

---

## 📈 Results

* Random Forest performed better than Linear Regression
* Achieved high accuracy with R² score of **0.89**
